Zadanie 58 - Pseudoklasy

  1. Co to są pseudoklasy?

    2. Pseudoklasy są to specjalne selektory CSS, które pozwalają na zastosowanie stylów do elementów w zależności od ich stanu lub położenia w dokumencie. Przykłady pseudoklas to :hover, :active, :focus czy :first-child.

  2. Uwaga dotycząca dwukropka:

    3. Dwukropek w CSS jest stosowany do oddzielenia nazwy selektora od pseudoklasy lub właściwości CSS od wartości. Na przykład: p:hover { color: red; } oznacza, że gdy kursor znajdzie się nad elementem p, jego kolor zostanie zmieniony na czerwony.

  3. Tabela opisującą pseudoklasy:
    4. Pseudoklasa Opis
    :hover Styl zastosowany, gdy kursor znajduje się nad elementem
    :active Styl zastosowany, gdy element jest aktywny (np. po kliknięciu)
    :focus Styl zastosowany, gdy element ma focus (np. po wybraniu pola formularza)
    :first-child Styl zastosowany do pierwszego dziecka elementu
    :last-child Styl zastosowany do ostatniego dziecka elementu
  4. Zastosowanie pseudoklasy :active:

    5. Pseudoklasa :active jest używana do zastosowania stylów do elementu w momencie, gdy jest on aktywny, np. po kliknięciu na przycisk. Przykład:

    button:active { background-color: blue; }
  5. Przykłady kodów definiowania pseudoklas:
    • :hover - a:hover { text-decoration: underline; }
    • :first-child - p:first-child
